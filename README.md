# ecProxy
An Easy to Handle A Proxy Service

## All in One Install
```shell
# Install node
bash <(curl https://raw.githubusercontent.com/Luphia/ecProxy/master/shell/install.sh -kL)
```

## Deploy with Source Code
```shell
git clone https://github.com/Luphia/ecProxy
cd ecProxy
npm install
npm start
```
